# 🛒 CV Store

A full-stack MERN E-Commerce platform with modern features including authentication, Google login, admin dashboard, wishlist, reviews, order management, and profile management.

## 🚀 Features

### User Features

* User Signup & Login
* Google Authentication
* Profile Management
* Product Search
* Product Details
* Add to Cart
* Wishlist
* Product Reviews & Ratings
* Order Placement
* Order History
* Forgot Password (OTP Email)

### Admin Features

* Admin Authentication
* Dashboard Analytics
* Product Management (CRUD)
* Order Management
* User Management
* Revenue Tracking

## ⚙️ Installation

### Clone Repository

```bash
git clone https://github.com/YOUR_USERNAME/CV-Store.git
```

### Client

```bash
cd client
npm install
npm run dev
```

### Server

```bash
cd server
npm install
npm run dev
```

---

## 🔑 Environment Variables

Create `.env` inside server:

```env
PORT=5000
MONGO_URI=YOUR_MONGODB_URI

JWT_SECRET=YOUR_SECRET_KEY

EMAIL=YOUR_EMAIL
EMAIL_PASSWORD=YOUR_PASSWORD

CLOUDINARY_CLOUD_NAME=YOUR_CLOUD_NAME
CLOUDINARY_API_KEY=YOUR_API_KEY
CLOUDINARY_API_SECRET=YOUR_API_SECRET

GOOGLE_CLIENT_ID=YOUR_GOOGLE_CLIENT_ID
GOOGLE_CLIENT_SECRET=YOUR_GOOGLE_CLIENT_SECRET
```
